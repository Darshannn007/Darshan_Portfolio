# FUTURE_FS_01 – Personal Portfolio Website

A modern and responsive developer portfolio website built using React and Vite.

## 🚀 Features

- Responsive Design
- Smooth Animations
- Projects Showcase
- Skills Section
- Contact Form using EmailJS
- Resume Download
- Fast Performance with Vercel Deployment

## 🛠️ Tech Stack

- React.js
- Vite
- Tailwind CSS
- JavaScript
- EmailJS
- Vercel

## 📂 Project Structure

```bash
src/
components/
assets/
pages/
public/
```

## ⚡ Installation

Clone the repository:

```bash
git clone <your-repo-link>
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
